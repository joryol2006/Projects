The Dataset

This is a classic data science dataset which consists of 32 unique automobiles with 11 different explanatory variables.
All Explanatory variables are numeric (continuous) variables.

Each Variable is listed below:
mpg = Miles/(US) gallon
cyl = Number of cylinders
disp = Displacement (cu.in.)
hp = Gross horsepower
drat = Rear axle ratio
wt = Weight (1000 lbs)
qsec = 1/4 mile time
vs = Engine (0 = V-shaped, 1 = straight)
am = Transmission (0 = automatic, 1 = manual)
gear = Number of forward gears
carb = Number of carburetors

Correlation Plot

From looking at the correlation plot, we can visually see the if each pair of variables are linearly related. The blue circles represent a positive correlation while the red circles represent a negative correlation. The darker the color, the stronger the correlation is between the variables.


The Displacement (disp) variable has a very strong positive correlation with Gross Horsepower (hp)


The Miles Per Gallon (mpg) variable has a very strong negative correlation with Weight (wt).

Finding Number of Optimal Clusters

NbClust package is used for determining the number of clusters and proposes to user the best clustering scheme from the different results obtained by varying all combinations of number of clusters, distance measures, and clustering methods.

The optimal number of clusters is 3

Plotting the Three Clusters

First step is to take calculate the distance matrix computed by using the specified distance measure to compute the distances between the rows of a data.

Second step is to perform the hierarchical cluster analysis that uses a set of dissimilarities for the n objects being clustered. In this case, 32 objects.

Initially, each object is assigned to its own cluster and then the algorithm proceeds iteratively, at each stage joining the two most similar clusters, continuing until there is just a single cluster.

At each stage distances between clusters are recomputed by the Lance–Williams dissimilarity update formula according to the particular clustering method being used.

Below is a cluster dendrogram that groups each car into three clusters based on the hierarchical cluster analysis.

d <- dist(mtcars_df)
h <- hclust(d)

plot(h)
rect.hclust(h, k = 3, border = 'red')
